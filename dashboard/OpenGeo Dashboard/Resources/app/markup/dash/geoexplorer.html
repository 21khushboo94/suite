GeoExplorer

GeoExplorer is a JavaScript user interface framework for building browser-based geospatial applications with the look, feel, and functionality of desktop GIS clients.

GeoExplorer allows you to quickly assemble rich mapping applications from multiple OGC Web Map Services (WMS), and embed your map in any map application or web page. GeoExplorer leverages GeoExt's capabilities to allow the display and management of layers, the ability to query and search across these layers, and the display of results.

Further information is available in the GeoExplorer Documentation.
