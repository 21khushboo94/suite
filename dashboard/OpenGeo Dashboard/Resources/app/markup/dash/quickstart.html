Quickstart

The OpenGeo Suite allows you to style, compose, and publish maps with ease. The following is a quick overview of the Suite's tools. For further documentation, see the Getting Started guide. For quick code examples using the OpenGeo Suite see the OpenGeo Recipe Book

Step 1: Load your data
  • Use the Layer Importer to load a collection of shapefiles into GeoServer. Each shapefile will become an individual map layer.
  • If you're looking for data to get started, the U.S. Census Bureau has free shapefiles available for download.
Step 2: Style your data
  • Use Styler to design the look and functionality for each map layer. Default styles are given, but can be modified to suit your needs.
Step 3: Compose your map
  • Use GeoExplorer to create a map comprised of one or several map layers. Adjust zoom and pan to customize your map's view.
Step 4: Publish your map
  • Use GeoExplorer's Publish Map function to save your map and embed it in another application or web page.
