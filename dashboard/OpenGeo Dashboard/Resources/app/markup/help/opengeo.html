About OpenGeo

OpenGeo is a social enterprise building the best open source web-based geospatial technology.

OpenGeo competes in the marketplace like a traditional software vendor while remaining true to its mission of making geospatial information more open. We build publicly available and accessible geospatial tools on compelling platforms that people want to use.

Our software makes it easy for any organization to get their geospatial information on the web using Open Geospatial Consortium (OGC) standards whether it is in a legacy system or our cutting edge open source software stack.
