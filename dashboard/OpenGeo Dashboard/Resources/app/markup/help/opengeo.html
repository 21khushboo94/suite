About the OpenGeo Suite

The OpenGeo Suite is the complete, OGC standards-compliant web mapping platform for robust, scalable deployments. Built on powerful, cutting-edge open source geospatial components, the OpenGeo Suite incorporates wide database and raster format support and strong Google, ArcSDE and Oracle connectivity.The OpenGeo Suite delivers functionality and flexibility for enterprises large and small.
