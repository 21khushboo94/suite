Getting Started

The following is a quick overview of the Suite's tools. For further documentation, please see the Getting Started guide.

Step 1: Load your data

  • Use the Layer Importer to load a collection of shapefiles or database tables into GeoServer. Each shapefile or database table will become an individual map layer.
  • If you are looking for data to get started, the U.S. Census Bureau has free shapefiles available for download.

Step 2: Style your data

  • Use Styler to design the look and feel for each map layer. The Importer will create default styles, but these can be modified to suit your needs.

Step 3: Compose your map

  • Use GeoExplorer to create a map comprised of one or several map layers. Adjust zoom and pan to customize your map's view.

Step 4: Publish your map

  • Use GeoExplorer's Publish Map function to save your map and embed it in another application or web page.

Advanced: Make your own mapping application

  • Look at the Recipe Book to learn how to make your own customized mapping applications using the OpenGeo Suite.
