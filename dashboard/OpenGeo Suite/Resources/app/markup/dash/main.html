Welcome to the OpenGeo Suite!

The OpenGeo Suite is a free, fully-featured web mapping environment that's easy to install, configure, and use. Use this dashboard to directly launch and learn about the primary features of the Suite. These include GeoServer, a map and feature server; GeoExplorer, a map publishing application; and Styler, an graphical interface for map design.
