Styler

STYLER IS GREAT. I NEED ONE LINE.

Styler makes it possible for anyone to make a map look the way they want it to. By abstracting traditional GIS styling components, with an easy to use interface, designing maps doesn't hinge upon tedious XML coding. Rather, Styler's framework in conjunction with the Styled Layer Descriptor (SLD) XML language, provides rapid prototyping and viewing of styles. Along with Styler's emphasis on open standards and flexible design, designing the map you want becomes easy and more efficient.

Styler is a web-based application built entirely with Javascript, leveraging the OpenLayers and GeoExt libraries. It is rendered with the Web Mapping Service (WMS) standard, which has extensions to render remote SLDs, and uses REST calls to persist styles to servers (please fix these last 2 sentences).

Styler is currently under development and will eventually be a set component of the GeoExplorer application.
