YSLD Tutorial
=============

The YSLD tutorial provides a set of steps that new users can follow to style a map using YSLD.

For this tutorial, you will need to download the following files from the `Natural Earth dataset <http://www.naturalearthdata.com/>`_:

 * 10 m cultural - `Populated Places <http://www.naturalearthdata.com/http//www.naturalearthdata.com/download/10m/cultural/ne_10m_populated_places.zip>`_
 * 10 m cultural - `Roads <http://www.naturalearthdata.com/http//www.naturalearthdata.com/download/10m/cultural/ne_10m_roads.zip>`_
 * 10 m cultural - `Countries <http://www.naturalearthdata.com/http//www.naturalearthdata.com/download/10m/cultural/ne_10m_admin_0_countries.zip>`_

Also download the following file from the `Natural Earth III extra data <http://www.shadedrelief.com/natural3/pages/extra.html>`_:

 * Terrestrial elevation data - `Large grayscale DEM <http://naturalearth.springercarto.com/ne3_data/dem_large.zip>`_

These four files should be imported into composer as new layers. The following pages will demonstrate how to style each of these layers to produce a useable map:
