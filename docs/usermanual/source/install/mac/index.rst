Installing Boundless Server on Mac OS X
=======================================

This section describes how to install or upgrade Boundless Server on Mac OS X. Please select the type of install below.

.. note:: Installers are no longer available for Mac OS X. To install the latest version of Boundless Server, you will need to convert your installation to either a WAR-based install or a virtual machine.
