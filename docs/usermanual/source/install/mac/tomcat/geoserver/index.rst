GeoServer Installation
======================

GeoServer is a primary component of Boundless Suite. This section will detail how to install GeoServer from a WAR file in Tomcat.

.. note:: During installation we will be editing text files that require Administrator access to modify. We recommend using :guilabel:`TextWrangler` (from `Apple App Store <https://itunes.apple.com/ca/app/textwrangler/id404010395?mt=12/>`__ ).
