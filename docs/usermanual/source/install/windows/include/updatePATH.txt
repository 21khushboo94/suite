

*On Windows 2012 Server*

       1. Open the Start menu and type :guilabel:`system environment`
       2. Select :guilabel:`Edit the system environment variables`

        .. figure:: /install/windows/include/edit_system_enviro.png

       3. Click :guilabel:`Environment Variables`

         .. figure:: /install/windows/include/click_enviro_var.png

       4. In the :guilabel:`System Variables` section, select :guilabel:`Path`
       5. Press :guilabel:`Edit...`

        .. figure:: /install/windows/include/edit_path.png

       6. Add the new directory (from above) to the beginning of the :guilabel:`Path` :guilabel:`Variable value`, and ensure you end it with a :guilabel:`semicolon`

       .. figure:: /install/windows/include/add_path.png

       8. Press :guilabel:`OK` and then :guilabel:`OK` again
       
*On Windows 10*     
   
       1. Open the Start menu and type :guilabel:`system environment`
       2. Select :guilabel:`Edit the system environment variables`
       
        .. figure:: /install/windows/include/edit_system_enviro.png
       
       3. Click :guilabel:`Environment Variables`
       
         .. figure:: /install/windows/include/click_enviro_var.png
       
       4. In the :guilabel:`System Variables` section, select :guilabel:`Path`
       5. Press :guilabel:`Edit...`
       
        .. figure:: /install/windows/include/edit_path.png

       6. Press :guilabel:`New`
       7. Type in the new directory (from above) 

	.. figure:: /install/windows/include/win10_edit.png

       8. Press :guilabel:`OK` and then :guilabel:`OK` again
