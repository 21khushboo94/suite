

*On Windows 2012 Server*

       #. Open the Start menu and type :guilabel:`system environment`
       #. Select :guilabel:`Edit the system environment variables`
       
        .. figure:: /install/windows/include/edit_system_enviro_var.png
       
       #. Click :guilabel:`Environment Variables`
       
         .. figure:: /install/windows/include/click_enviro.png
       
       #. In the :guilabel:`System Variables` section, select :guilabel:`Path`
       #. Press :guilabel:`Edit...`
       
        .. figure:: /install/windows/include/edit_path.png
       
       #. Add the new directory (from above) to the beginning of the :guilabel:`Path` :guilabel:`Variable value`, and ensure you end it with a :guilabel:`semicolon`
       
       .. figure:: /install/windows/include/add_path.png
       
       #. Press :guilabel:`OK` and then :guilabel:`OK` again
       
*On Windows 10*     
   
      #. Open the Start menu and type :guilabel:`system environment`
      #. Select :guilabel:`Edit the system environment variables`
       
        .. figure:: /install/windows/include/edit_system_enviro_Var.png
       
       #. Click :guilabel:`Environment Variables`
 
       #. In the :guilabel:`System Variables` section, select :guilabel:`Path`
       #. Press :guilabel:`Edit...`
       
        .. figure:: /install/windows/include/edit_path.png

       #. Press "New"
       #. Type in :file:`c:\\libjpeg-turbo`
       #. Press "OK" and then "OK" again
