OpenGeo Suite for Ubuntu Linux
==============================

This document describes how to manage installaton of OpenGeo Suite for Ubuntu Linux.

.. note:: QGIS, while part of OpenGeo Suite, is not currently bundled as a package by Boundless. To use QGIS with an Ubuntu system, please see the `QGIS community installation instructions <https://www.qgis.org/en/site/forusers/download.html>`_.

Please select the option that best matches your system:

:ref:`intro.installation.ubuntu.install`
  For new installations.

:ref:`intro.installation.ubuntu.minorupdate`  
  For minor version updates. To update from OpenGeo Suite 4.x, start here.

:ref:`intro.installation.ubuntu.majorupdate`
  For major version updates. To update from OpenGeo Suite 3.x, start here.

:ref:`intro.installation.ubuntu.upgrade`
  For upgrading to OpenGeo Suite Enterprise.

:ref:`intro.installation.ubuntu.postinstall`
  Useful information about the installation.

:ref:`intro.installation.ubuntu.packages`
  List of packages and their functions.

:ref:`intro.installation.ubuntu.uninstall`
  For removing OpenGeo Suite packages.
