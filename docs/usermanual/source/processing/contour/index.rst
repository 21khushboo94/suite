Creating a contour map with a WPS process
=========================================

This tutorial will show how to create a contour map from a raster data layer. It will show this using two methods: creation of a static vector contour layer, and creation of a dynamic contour style using :ref:`cartography.rt`. Both methods employ a WPS process.
