GeoScript
=========

GeoScript is a scripting library that adds spatial capabilities to dynamic scripting languages.  Through GeoScript, one can access low-level GeoTools functions without needing to know Java.

GeoScript is included in the OpenGeo Suite, with bindings for both JavaScript and Python.
