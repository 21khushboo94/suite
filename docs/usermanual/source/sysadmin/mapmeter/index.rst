Using Mapmeter
==============

Mapmeter is a cloud-based service to enable you to monitor the usage and health of GeoServer.

It is available as a two-week free trial to anyone, and as well is available for free to Enterprise clients. If you are interested in becoming an Enterprise client, please  `contact us <http://boundlessgeo.com/about/contact-us/sales/>`_.
