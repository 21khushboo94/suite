Glossary
========

.. glossary::

   GetCapabilities
     WMS request that returns a capabilities document, a list of all layers and functions supported by the WMS.  GeoExplorer reads the capabilities document of a server in order to determine the available layers.

   GetFeatureInfo
     WMS request that polls a given map location and typically returns a list of attributes and values for the data at that location.

   GetLegendGraphic
     WMS request that returns a graphic rendered with the style of a given layer.   

   GeoExt
     A JavaScript toolkit for creating rich web mapping applications.  GeoExt is built using OpenLayers and ExtJS.  Learn more at `geoext.org <http://geoext.org>`_.

   OGC
     The `Open Geospatial Consortium <http://www.opengeospatial.org/>`_ (OGC) is a standards organization that develops specifications for geospatial services.

   Styler
     A web-based graphical SLD editor, designed by the same team as GeoExplorer.

   WMS
     The `Web Map Service <http://www.opengeospatial.org/standards/wms>`_ (WMS) specification from the :term:`OGC` defines an interface for requesting rendered map images across the web.  WMS can refer either to the protocol itself or a server that understands that protocol.
