GeoExplorer¶

Quickly assemble rich mapping applications from multiple OGC Web Mapping Services (WMS) with GeoExplorer. Using the publish wizard, embed your mapping application in any web page.

Quick Start¶

If you’re interested in diving right in, follow the steps below. For more detail on getting started with GeoExplorer, see the introductory documentation.

  1. Open up GeoExplorer in your browser.
  2. Use the add and remove buttons to choose which layers will be included in your application.
  3. From the “Available Layers” dialog, add new WMS servers (e.g. http://terraserver-usa.com/ogccapabilities.ashx).
  4. Reorder layers by dragging them around. Only one layer at a time can be visible in the “Base Layers” group.
  5. Launch the “Publish Map” wizard to embed your application in any web page.

For a more detailled description of what you can do with GeoExplorer, see the Working With GeoExplorer documentation.
