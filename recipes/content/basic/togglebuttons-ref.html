Toggle Buttons Documentation

When associating multiple buttons with a map, only one button can be active at a time. ExtJS makes it very simple to ensure that only one button from a group is active at a time. By identifying buttons into groups, and then using the Button toggleGroup property, only one button can be active at a time. Our recipe toggles between an area and length buttons.
This recipe uses the following classes:
  • GeoExt.MapPanel
  • OpenLayers.Layer.WMS
  • OpenLayers.Control.Measure
  • OpenLayers.Handler.Path
  • OpenLayers.Handler.Polygon
