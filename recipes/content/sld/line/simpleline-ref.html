Simple Line Documentation

This example specifies lines be colored black with a thickness of 3 pixels.
There is one in one for this SLD, which is the simplest possible situation. (All subsequent examples will contain one and one unless otherwise specified.) Styling lines is accomplished via the (lines 3-8). Line 5 specifies the color of the line to be black (#000000), while line 6 specifies the width of the lines to be 3 pixels.
