Simple Point Documentation

This example specifies points be styled as red circles with a diameter of 6 pixels.
There is one in one for this SLD, which is the simplest possible situation. (All subsequent examples will contain one and one unless otherwise specified.) Styling points is accomplished via the (lines 3-13). Line 6 specifies the shape of the symbol to be a circle, with line 8 determining the fill color to be red (#FF0000). Line 11 sets the size (diameter) of the graphic to be 6 pixels.
